# Frontend (Vite + React)

## Запуск

В папке `frontend`:

```powershell
npm install
copy .env.example .env
npm run dev
```

Открой: `http://localhost:5173`
Открой: `http://localhost:3000`

## Настройка API

По умолчанию фронт ходит в `http://localhost:8001`.
Можно поменять в `frontend/.env`:

```
VITE_API_BASE=http://localhost:8001
```
